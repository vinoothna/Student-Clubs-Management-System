ALL ABOUT IT!

CREWS - Student Club Management System allows an individual to join or leave a club. One can register themselves for events organized. It always keeps you updated with the posts and show you the notifications about events. Each club has an admin who can organize events and conduct polls. The Super Admin of the application can only add / remove clubs.
